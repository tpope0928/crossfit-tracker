Crossfit Tracker APP

This is a sinatra based MCV app for logging crossfit workouts. Users can create accounts and then log workouts with the workout name, content, time completed, and rounds completed. The user can then view, edit, and delete those workouts.

Install Instructions: Clone and Run bundle install
